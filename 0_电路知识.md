# 信号线

传递信号的电路

# 信号线的悬空

信号线在电路中既没有连接到高电平（如VCC或电源正极），也没有连接到低电平（如GND或电源负极）的状态

## pull-up resistor上拉电阻

电源Vcc到引脚上的电阻。作用是在平时把该引脚钳制在高电平

## pull-down resistor 下拉电阻

GND到引脚的电阻。作用是在平时把该引脚钳制在低电平。
